# E-Commerce Web Application - dimo

This is a modern e-commerce web application built using React, Vite, Tailwind CSS, and various supporting libraries. The project is designed to provide a responsive, user-friendly, and scalable platform for online shopping.

---

## Features

- **React**: Component-based architecture for reusable and efficient UI development.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Redux Toolkit**: State management with minimal boilerplate.
- **React Router DOM**: Seamless navigation with routing.
- **Material UI**: Prebuilt components for consistent design.
- **Formik & Yup**: Form handling and validation.
- **React Toastify**: User-friendly toast notifications.
- **Axios**: Simplified HTTP requests.

---

## Installation and Setup

To set up the project locally:

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd <repository-folder>
   ```
2. Install dependencies:
   npm install

3. Start the development server:
   npm run dev

4. Build the production-ready version:
   npm run build

5. Preview the production build:
   npm run preview

- **src/**: Contains all the source code, including components, pages, and styles. -**public/**: Static assets like images. -**vite.config.js**: Configuration file for Vite. -**tailwind.config.js**: Custom Tailwind CSS configuration.
